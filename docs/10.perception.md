# 10.机器人感知

机器人与环境交互的基础是**感知**。通过各类传感器，机器人能够获取周围环境的物理量信息，如光线、距离、声音等，这些原始数据经过处理后转化为机器人可理解的 “环境描述”，为决策与控制提供依据。

在本章中，我们将从**机器人如何感知世界**这一核心问题出发，系统介绍ROS 2中与传感器相关的消息类型及其应用场景，并通过一个实战案例——基于摄像头的绿色色块检测，体验机器人视觉感知的完整开发流程。

本章内容将涵盖：

+ ROS 2 常用传感器消息类型（sensor_msgs）；
+ 使用 OpenCV 读取摄像头视频流；
+ 实时检测画面中的绿色色块，计算其中心坐标与面积；
+ 将检测结果发布为 ROS 2 话题；
+ 使用命令行工具与 RViz2 验证感知结果。

通过本章的学习，你将掌握如何在 ROS 2 系统中实现从`传感器数据读取` → `感知处理` → `信息发布` → `可视化展示`的完整流程。

## 10.1 机器人感知概述

机器人感知（Perception）是指机器人通过传感器获取外部环境信息并进行理解的过程。在典型的机器人系统中，感知模块承担了“眼睛”和“耳朵”的角色，其输出数据直接影响后续的决策与控制。

常见传感器包括：

|传感器类型	|功能	|对应消息类型（sensor_msgs）|	示例话题名|
|--|--|--|--|
|摄像头	|获取环境图像	|sensor_msgs/Image	|/camera/image_raw|
|激光雷达|	获取二维距离信息|	sensor_msgs/LaserScan|	/scan|
|IMU|	获取加速度与角速度|	sensor_msgs/Imu|	/imu/data|
|超声波|	测距|	sensor_msgs/Range|	/ultrasonic/front|
|GPS|	位置信息|	sensor_msgs/NavSatFix|	/gps/fix|

在 ROS 2 中，[sensor_msgs](https://docs.ros2.org/foxy/api/sensor_msgs/index-msg.html) 是一个标准消息包，定义了机器人常用传感器的消息接口。熟悉这些接口，是理解机器人系统感知模块的基础。

可使用 `ros2 interface  show <包名>/msg/<消息名>` 命令查看具体消息定义，如：

`$ ros2 interface show sensor_msgs/msg/Image`

此时会显示`Image`（图像消息）定义：

```python
# This message contains an uncompressed image
# (0, 0) is at top-left corner of image

std_msgs/Header header # Header timestamp should be acquisition time of image
	builtin_interfaces/Time stamp
		int32 sec
		uint32 nanosec
	string frame_id
                             # Header frame_id should be optical frame of camera
                             # origin of frame should be optical center of cameara
                             # +x should point to the right in the image
                             # +y should point down in the image
                             # +z should point into to plane of the image
                             # If the frame_id here and the frame_id of the CameraInfo
                             # message associated with the image conflict
                             # the behavior is undefined

uint32 height                # image height, that is, number of rows
uint32 width                 # image width, that is, number of columns

# The legal values for encoding are in file src/image_encodings.cpp
# If you want to standardize a new string format, join
# ros-users@lists.ros.org and send an email proposing a new encoding.

string encoding       # Encoding of pixels -- channel meaning, ordering, size
                      # taken from the list of strings in include/sensor_msgs/image_encodings.hpp

uint8 is_bigendian    # is this data bigendian?
uint32 step           # Full row length in bytes
uint8[] data          # actual matrix data, size is (step * rows)
```

### (1)视觉传感器（摄像头）

视觉传感器是最常用的机器人传感器之一，通过采集图像获取环境的二维视觉信息，广泛应用于目标检测、识别、导航等场景。

+ 核心消息类型：`sensor_msgs/Image`
该消息用于传输未压缩的图像数据，包含以下关键字段：
    + `header`：包含时间戳（stamp）和坐标系（frame_id），用于时间同步和空间定位；
    + `height/width`：图像的高度和宽度（像素）；
    + `encoding`：图像编码格式，如bgr8（OpenCV 默认格式）、rgb8（RGB 格式）、mono8（单通道灰度图）等；
    + `data`：图像像素数据，以字节数组形式存储。

+ 辅助消息类型：`sensor_msgs/CompressedImage`
    + 用于传输压缩后的图像（如 JPEG、PNG），适用于带宽有限的场景，核心字段为format（压缩格式）和data（压缩数据）。

### (2)激光雷达（LiDAR）

激光雷达通过发射激光束并接收反射信号，计算与障碍物的距离，可生成环境的二维或三维点云数据，常用于 SLAM（同步定位与地图构建）、避障等。

+ 2D 激光雷达消息：`sensor_msgs/LaserScan`
    + `header`：时间戳和坐标系；
    + `angle_min/angle_max`：扫描角度范围（弧度）；
    + `angle_increment`：相邻两个激光点的角度差（弧度）；
    + `range_min/range_max`：有效测距范围（米）；
    + `ranges`：距离数组，每个元素对应一个角度的测距值（米，超出范围时为inf）；
    + `intensities`：激光反射强度数组（部分雷达支持）。

+ 3D 激光雷达消息：`sensor_msgs/PointCloud2`
用于传输三维点云，包含每个点的三维坐标（x, y, z）及可能的附加信息（如反射强度、颜色）。该消息结构较复杂，需通过pcl等库解析。

### (3)惯性测量单元（IMU）

IMU 通过加速度计和陀螺仪测量机器人的线性加速度和角速度，常用于姿态估计（如倾斜角、旋转角度）。

+ 核心消息类型：`sensor_msgs/Imu`
    + `header`：时间戳和坐标系；
    + `linear_acceleration`：三维线性加速度（m/s²）；
    + `angular_velocity`：三维角速度（rad/s）；
    + `orientation`：四元数表示的姿态（可选，需结合磁力计校准后输出）。


## 10.2 案例：视觉感知——绿色色块检测

在本节中，我们将实现一个基础的视觉感知功能：通过笔记本摄像头捕获实时画面，检测其中的绿色色块，计算其中心坐标与面积，并将结果发布到ROS2话题中。

### 目标

+ 读取摄像头视频流；
+ 检测画面中绿色色块；
+ 计算色块中心点坐标与面积；
+ 将检测结果以话题形式发布；
+ 在命令行和 RViz2 中查看结果。

### (1) 环境准备

+ ROS 2 Humble（已完成）；
+ OpenCV视觉库安装：
    `$ sudo apt install python3-opencv`
+ 摄像头设备：
    + 若使用虚拟机，请在虚拟机菜单栏选择"设备"-> "摄像头"，将电脑的摄像头加载到虚拟机中
    + 确保宿主机摄像头已共享到虚拟机设备中（通常为`/dev/video0`）
    + 可执行此命令查看是否存在: `$ ls /dev/`

### (3) 编写节点：color_detector.py

```python
import rclpy
from rclpy.node import Node
from std_msgs.msg import String
import cv2
import numpy as np

class ColorDetector(Node):
    def __init__(self):
        super().__init__('color_detector')
        # 发布者:发布检测结果信息(坐标及面积)
        self.publisher_ = self.create_publisher(String, 'color_info', 10)
        # 打开摄像头（0 代表 /dev/video0）
        self.cap = cv2.VideoCapture(0)
        if not self.cap.isOpened():
            self.get_logger().error("无法打开摄像头")
            return
        # 创建定时器,周期性(0.2秒)读取相机图片并处理
        self.timer = self.create_timer(0.2, self.timer_callback)

    # 定时器函数
    def timer_callback(self):
        # 从摄像头读取图片
        ret, frame = self.cap.read()
        if not ret:
            self.get_logger().warn("读取摄像头帧失败")
            return
        # 转换为HSV空间
        hsv = cv2.cvtColor(frame, cv2.COLOR_BGR2HSV)
        # 绿色的HSV范围 H: 35-85, S: 50-255, V: 50-255
        lower_green = np.array([35, 50, 50])
        upper_green = np.array([85, 255, 255])
        # 颜色区域掩码
        mask = cv2.inRange(hsv, lower_green, upper_green)        
        # 计算颜色区域轮廓
        contours, _ = cv2.findContours(mask, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
        cx, cy = -1, -1 # 颜色区域质心坐标
        area = 0 # 颜色区域面积
        if contours: # 如果检测到了颜色区域
            c = max(contours, key=cv2.contourArea) # 获取最大一块区域
            area = cv2.contourArea(c) #计算区域面积
            # 计算颜色区域质心坐标
            M = cv2.moments(c)
            if M["m00"] != 0:
                cx = int(M["m10"] / M["m00"]) # 质心X坐标
                cy = int(M["m01"] / M["m00"]) # 质心Y坐标
                # 绘制结果
                cv2.circle(frame, (cx, cy), 5, (0, 0, 255), -1)
                cv2.putText(frame, f"{cx}, {cy}, {area}", (cx+10, cy),
                            cv2.FONT_HERSHEY_SIMPLEX, 0.5, (0, 0, 255), 1)
        
        # 将颜色区域坐标及面积转为字符串，并发布
        msg = String()
        msg.data = f"{cx}, {cy}, {area}"
        self.publisher_.publish(msg)
        
        # 绘制结果
        cv2.imshow("Color mask", mask)
        cv2.imshow("Color Detector", frame)
        if cv2.waitKey(1) & 0xFF == ord('q'):
            self.get_logger().info("检测结束")
            self.destroy_node()
            cv2.destroyAllWindows()

def main(args=None):
    rclpy.init(args=args)
    node = ColorDetector()
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    node.cap.release()
    cv2.destroyAllWindows()
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### (4) 修改`setup.py`
