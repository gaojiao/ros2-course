# 3.Python基础

本课程使用Python语言进行ROS2程序的开发，因此，本章复习Python编程的核心基础知识，包括基本语法、数据类型、控制结构、函数及模块的使用，并简要介绍面向对象编程中类的基础用法。

## 3.1 Python 基本语法

### 3.1.1 注释

注释是代码中不被执行的说明性文字，用于提高代码可读性。Python 支持两种注释方式：

+ 单行注释：以`#`开头，`#`后的内容均为注释

    ```python
    # 这是一行单行注释
    a = 10  # 行尾的注释
    ```

+ 多行注释：用三个单引号`'''`或三个双引号`"""`包裹

    ```python
    """
    这是多行注释
    可以跨越多行
    """
    ```

### 3.1.2 变量与赋值

Python 是动态类型语言，变量无需声明类型，直接通过赋值创建：

```python
x = 5  # 整数
name = "Alice"  # 字符串
is_student = True  # 布尔值
```

+ 变量名（标识符）规则：由字母、数字、下划线组成，不能以数字开头，区分大小写（如num与Num是不同变量）

+ 赋值特性：支持多变量同时赋值
    ```python
    a, b, c = 1, 2, 3  # 多变量赋值
    x = y = 10  # 链式赋值
    ```

### 3.1.3 缩进

Python 通过缩进来区分代码块（无需`{}`），通常使用 4 个空格作为缩进（推荐）：

```python
if x > 0:
    print("正数")  # 缩进表示属于if代码块
else:
    print("非正数")
```

*注意：缩进量要一致，否则会导致IndentationError*

### 3.2 数据类型

Python 提供丰富的数据类型，可分为基本类型和复合类型。


### 3.2.1 基本类型

+ 整数（int）：无大小限制的整数，如-3、0、100
+ 浮点数（float）：带小数点的数字，如3.14、-0.5
+ 字符串（str）：由字符组成的序列，用单引号`'`或双引号`"`包裹，支持切片（`s[0:3]`取前 3 个字符）
    ```python
    s = "Hello"
    print(s[1:4])  # 输出 'ell'（从索引1到3，左闭右开）
    print(s.upper())  # 输出 'HELLO'（字符串方法）
    ```
+ 布尔值（bool）：仅True（真）和False（假），常用于条件判断

### 3.2.2 复合类型


+ 列表（list）：有序、可修改的元素集合，用[]表示
    ```python
    nums = [1, 2, 3, 4]
    nums.append(5)  # 添加元素，结果为 [1,2,3,4,5]
    nums[0] = 0  # 修改元素，结果为 [0,2,3,4,5]
    ```

+ 元组（tuple）：有序、不可修改的元素集合，用()表示
    ```python
    point = (3, 4)  # 不可修改，若尝试point[0]=5会报错
    ```

+ 字典（dict）：无序的键值对集合，用{}表示，键唯一
    ```python
    student = {"name": "Bob", "age": 20}
    print(student["name"])  # 输出 'Bob'
    student["age"] = 21  # 修改值
    ```

+ 集合（set）：无序、无重复元素的集合，用{}或set()创建
    ```python
    s = {1, 2, 2, 3}  # 自动去重，结果为 {1,2,3}
    ```

### 3.2.3 类型转换

通过内置函数可实现类型转换：

```python
int("123")  # 字符串转整数，结果 123
float(5)    # 整数转浮点数，结果 5.0
str(3.14)   # 浮点数转字符串，结果 "3.14"
```

## 3.3 控制结构

控制结构用于控制代码执行流程，包括条件语句和循环语句。

### 3.3.1 条件语句（if-elif-else）

根据条件执行不同代码块：

```python
score = 85
if score >= 90:
    print("优秀")
elif score >= 80:  # 多个条件分支
    print("良好")
else:
    print("继续努力")
```

### 3.3.2 循环语句

+ for 循环：用于遍历序列（列表、字符串等）
    ```python
    fruits = ["apple", "banana", "cherry"]
    for fruit in fruits:
        print(fruit)  # 依次输出每个水果
    ```

    配合range()生成数字序列：
    ```python
    for i in range(3):  # range(3) 生成 0,1,2
        print(i)
    ```

+ while 循环：当条件为True时重复执行
    ```python
    count = 0
    while count < 3:
        print(count)
        count += 1  # 避免无限循环
    ```

### 3.3.3 循环控制

+ break：终止当前循环
+ continue：跳过本次循环剩余部分，直接进入下一次

```python
for i in range(5):
    if i == 2:
        continue  # 跳过i=2的循环
    if i == 4:
        break     # 当i=4时终止循环
    print(i)  # 输出 0,1,3
```



### 3.4 函数

函数是封装的可重用代码块，通过def定义。

### 3.4.1 函数定义与调用

```python
# 定义函数：计算两数之和
def add(a, b):
    result = a + b
    return result  # 返回结果

# 调用函数
sum = add(3, 5)
print(sum)  # 输出 8
```

### 3.4.2 参数类型 

+ 位置参数：按参数顺序传递

+ 关键字参数：通过参数名指定，可打乱顺序
    ```python
    def greet(name, age):
        print(f"Hello {name}, you are {age} years old.")

    greet("Alice", 20)  # 位置参数
    greet(age=20, name="Alice")  # 关键字参数
    ```

+ 默认参数：定义时指定默认值，调用时可省略
    ```python
    def greet(name, age=18):  # age默认值为18
        print(f"Hello {name}, you are {age} years old.")

    greet("Bob")  # 输出 "Hello Bob, you are 18 years old."
    ```

### 3.4.3 匿名函数（lambda） 

简洁的单行函数，用lambda定义，仅能包含一个表达式：
```python
add = lambda a, b: a + b  # 等价于def add(a,b): return a+b
print(add(2, 3))  # 输出 5
```

## 3.5 模块与包

模块是包含 Python 代码的文件（.py），包是多个模块的集合（含__init__.py的文件夹），用于代码复用。

### 3.5.1 导入模块

```python
import math  # 导入整个模块
print(math.sqrt(16))  # 使用模块中的函数，输出4.0

from math import sqrt  # 仅导入指定函数
print(sqrt(25))  # 直接使用函数，输出5.0

import math as m  # 给模块起别名
print(m.pi)  # 输出圆周率3.141592653589793
```

## 3.6 类的基础使用

类是面向对象编程（OOP）的核心，用于封装数据（属性）和操作数据的方法（函数）。

### 3.6.1 类的定义

用class关键字定义类，类名通常首字母大写：

```python
# 定义一个学生类
class Student:
    # 初始化方法（构造函数），创建对象时自动调用
    def __init__(self, name, age, major):
        self.name = name    # 实例属性：每个对象独有的属性
        self.age = age
        self.major = major
    
    # 实例方法：操作实例属性，第一个参数必须是self（代表当前对象）
    def introduce(self):
        return f"我叫{self.name}, {self.age}岁, 专业是{self.major}"
    
    # 另一个实例方法
    def study(self, subject):
        return f"{self.name}正在学习{subject}"        
```

### 3.6.2 对象的创建与使用

类是"模板"，对象是类的具体实例：

```python
# 创建类的实例（对象）
stu1 = Student("张三", 20, "自动化")
stu2 = Student("李四", 21, "物理学")

# 访问属性
print(stu1.name)  # 输出 "张三"
print(stu2.major)  # 输出 "自动化"

# 调用方法
print(stu1.introduce()) # 输出 "我叫张三, 20岁, 专业是自动化"
print(stu2.study("高等数学")) # 输出 "李四正在学习高等数学"
```
### 3.6.3 类的继承

继承允许定义一个继承父类属性和方法的新类，用`class 子类(父类):`定义。

```python
# 父类（基类）
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def introduce(self):
        return f"我叫{self.name}, {self.age}岁"

# 子类1（派生类）
class Student(Person):  # 继承Person类
    def __init__(self, name, age, major):
        super().__init__(name, age)  # 调用父类的初始化方法
        self.major = major
    
    # 重写父类方法
    def introduce(self):
        return f"我叫{self.name}, {self.age}岁, 专业是{self.major}"
    
    # 子类特有的方法
    def study(self, subject):
        return f"{self.name}正在学习{subject}"

# 子类2（派生类）
class Teacher(Person):  # 继承Person类
    def __init__(self, name, age):
        super().__init__(name, age)  # 调用父类的初始化方法
    
    # 子类特有的方法
    def teach(self, subject):
        return f"{self.name}正在教{subject}课程"

# 使用子类
stu1 = Student("王五", 22, "数学")
print(stu1.introduce())  # 调用子类重写的方法
print(stu1.study("微积分"))

t1 = Teacher("张三", 40)
print(t1.introduce())  # 调用父类已有的方法
print(t1.teach('自动控制原理'))  # 调用子类重写的方法
```
