# 13.TF变换

在机器人学中，坐标变换是一个核心问题。一个机器人通常包含多个部件：底盘、激光雷达、摄像头、机械臂末端执行器等，每个部件都有自己的坐标系。我们需要知道这些坐标系之间的相对位置关系，才能进行感知、规划和控制。ROS 2 提供了 `tf2`（Transform Framework 2）库，专门用于管理坐标系变换。本章将介绍 `tf2` 的基本概念、使用方法，并通过 Python 代码实现坐标变换的广播与监听。

可参考ROS 2 官方`tf2`文档： https://docs.ros.org/en/rolling/Tutorials/Intermediate/Tf2/Tf2-Main.html

## 13.1 什么是TF变换？

TF（Transform）是一个让用户能够随时跟踪多个坐标系之间相对位置（平移和旋转）的系统。在 ROS 2 中，`tf2` 是第二代变换库，它提供了以下核心功能：

- **坐标系树（TF Tree）**：所有坐标系通过父-子关系构成一棵树，每个坐标系只有一个父坐标系，但可以有多个子坐标系。
- **广播器（Broadcaster）**：负责发布坐标系之间的变换关系（即从父坐标系到子坐标系的变换）。
- **监听器（Listener）**：负责接收所有广播的变换，并缓存它们，供用户查询任意两个坐标系之间的变换。
- **时间性**：每个变换都带有时间戳，支持查询历史变换（在给定时间点或时间段内的变换）。

在机器人系统中，常见的坐标系有：
- `map`：世界地图坐标系
- `odom`：里程计坐标系
- `base_link`（或 `base_footprint`）：机器人本体坐标系
- `laser`：激光雷达坐标系
- `camera`：摄像头坐标系

## 13.2 TF树与静态/动态变换

在 TF 树中，坐标系之间的变换可以是**静态**的或**动态**的。

- **静态变换**：表示两个坐标系之间的相对位置固定不变，例如激光雷达安装在机器人底盘上的固定位置。静态变换通常只需要发布一次，可以使用 `static_transform_publisher` 工具或编写静态广播器。
- **动态变换**：表示两个坐标系之间的相对位置随时间变化，例如移动机器人的底盘相对于世界坐标系的位置会随着运动而变化，或者机械臂的关节角度变化导致末端坐标系变化。动态变换需要持续发布。

ROS 2 提供了多个工具来查看和调试 TF 树：
- `ros2 run tf2_ros tf2_echo`：打印两个坐标系之间的变换。
- `ros2 run tf2_ros tf2_monitor`：监控 TF 树中所有坐标系的发布频率和延迟。
- `ros2 run tf2_tools view_frames`：生成 TF 树的 PDF 图（需要安装 `tf2_tools`）。

## 13.3 在 ROS 2 中使用 TF2（Python）

我们通过一个简单的例子来演示如何创建 TF 广播器和监听器。假设我们有一个机器人，其 `base_link` 坐标系上固定安装了一个激光雷达，激光雷达的坐标系为 `laser_frame`，它在 `base_link` 前方 0.2 米处，且绕 Z 轴旋转了 10 度（静态变换）。同时，机器人可以在世界坐标系中移动，我们通过模拟运动（例如键盘控制）来动态发布 `base_link` 相对于 `world` 坐标系的变换。

### 13.3.1 创建功能包

首先，创建一个新的功能包，依赖于 `rclpy`、`tf2_ros`、`geometry_msgs` 和 `std_msgs`。

```bash
cd ~/ros2_ws/src
ros2 pkg create my_tf_pkg --build-type ament_python --dependencies rclpy tf2_ros geometry_msgs std_msgs
```

### 13.3.2 静态变换广播器

静态变换可以使用 `tf2_ros` 提供的 `StaticTransformBroadcaster` 轻松实现。我们创建一个Python节点，发布激光雷达相对于 `base_link` 的静态变换。

在 `my_tf_pkg/my_tf_pkg/static_tf_broadcaster.py` 中写入以下代码：

```python
import rclpy
from rclpy.node import Node
from tf2_ros import StaticTransformBroadcaster
from geometry_msgs.msg import TransformStamped
import math

def quaternion_from_euler(ai, aj, ak):
    ai /= 2.0
    aj /= 2.0
    ak /= 2.0
    ci = math.cos(ai)
    si = math.sin(ai)
    cj = math.cos(aj)
    sj = math.sin(aj)
    ck = math.cos(ak)
    sk = math.sin(ak)
    cc = ci*ck
    cs = ci*sk
    sc = si*ck
    ss = si*sk
    q = [cj*sc - sj*cs,
         cj*ss + sj*cc,
         cj*cs - sj*sc,
         cj*cc + sj*ss]
    return q

class StaticTFBroadcaster(Node):
    def __init__(self):
        super().__init__('static_tf_broadcaster')
        self.broadcaster = StaticTransformBroadcaster(self)
        self.publish_static_transform()

    def publish_static_transform(self):
        t = TransformStamped()
        t.header.stamp = self.get_clock().now().to_msg()
        t.header.frame_id = 'base_link'          # 父坐标系
        t.child_frame_id = 'laser_frame'         # 子坐标系
        # 平移：在 base_link 前方 0.2 米（x 方向）
        t.transform.translation.x = 0.2
        t.transform.translation.y = 0.0
        t.transform.translation.z = 0.0
        # 旋转：绕 Z 轴旋转 10 度（使用四元数）
        angle = math.radians(10.0)
        q = quaternion_from_euler(0.0, 0.0, angle)
        t.transform.rotation.x = q[0]
        t.transform.rotation.y = q[1]
        t.transform.rotation.z = q[2]
        t.transform.rotation.w = q[3]

        self.broadcaster.sendTransform(t)
        self.get_logger().info('Published static transform from base_link to laser_frame')

def main(args=None):
    rclpy.init(args=args)
    node = StaticTFBroadcaster()
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

> 在ROS2中姿态用四元数(quaternion)表示，代码中的`quaternion_from_euler` 函数作用是将欧拉角转换为四元数。

### 13.3.3 动态变换广播器

动态变换广播器用于发布机器人位置随时间变化的变换。这里我们模拟一个简单的圆形运动：机器人以原点为中心，半径为 1 米，角速度 0.5 rad/s 运动。我们发布 `world` 到 `base_link` 的变换。

创建 `my_tf_pkg/my_tf_pkg/dynamic_tf_broadcaster.py`：

```python
import rclpy
from rclpy.node import Node
from tf2_ros import TransformBroadcaster
from geometry_msgs.msg import TransformStamped
import math
import time

class DynamicTFBroadcaster(Node):
    def __init__(self):
        super().__init__('dynamic_tf_broadcaster')
        self.broadcaster = TransformBroadcaster(self)
        self.timer = self.create_timer(0.1, self.broadcast_tf)  # 10 Hz
        self.start_time = self.get_clock().now()

    def broadcast_tf(self):
        t = TransformStamped()
        t.header.stamp = self.get_clock().now().to_msg()
        t.header.frame_id = 'world'            # 父坐标系
        t.child_frame_id = 'base_link'         # 子坐标系

        # 模拟圆形运动
        elapsed = (self.get_clock().now() - self.start_time).nanoseconds / 1e9
        radius = 1.0
        angular_speed = 0.5  # rad/s
        angle = angular_speed * elapsed
        x = radius * math.cos(angle)
        y = radius * math.sin(angle)
        z = 0.0

        t.transform.translation.x = x
        t.transform.translation.y = y
        t.transform.translation.z = z

        # 机器人朝向为运动方向（切线方向），这里简化：不设置旋转，即无旋转
        t.transform.rotation.x = 0.0
        t.transform.rotation.y = 0.0
        t.transform.rotation.z = 0.0
        t.transform.rotation.w = 1.0

        self.broadcaster.sendTransform(t)
        # 打印位置
        self.get_logger().info(f'Publishing transform: ({x:.2f}, {y:.2f})')

def main(args=None):
    rclpy.init(args=args)
    node = DynamicTFBroadcaster()
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### 13.3.4 TF 监听器

监听器用于查询任意两个坐标系之间的变换。这里我们创建一个节点，监听 `world` 到 `laser_frame` 的变换，并打印结果。

创建 `my_tf_pkg/my_tf_pkg/tf_listener.py`：

```python
import rclpy
from rclpy.node import Node
from tf2_ros import Buffer, TransformListener
from tf2_ros import TransformException
from geometry_msgs.msg import TransformStamped
import sys

class TFListener(Node):
    def __init__(self):
        super().__init__('tf_listener')
        self.tf_buffer = Buffer()
        self.tf_listener = TransformListener(self.tf_buffer, self)
        self.timer = self.create_timer(1.0, self.on_timer)  # 每秒查询一次

    def on_timer(self):
        try:
            # 查询从 world 到 laser_frame 的变换（当前时间）
            trans = self.tf_buffer.lookup_transform(
                'world',            # 目标坐标系
                'laser_frame',      # 源坐标系
                rclpy.time.Time()   # 当前时间（也可以指定具体时间）
            )
            self.get_logger().info(
                f'Transform: translation=({trans.transform.translation.x:.2f}, '
                f'{trans.transform.translation.y:.2f}, {trans.transform.translation.z:.2f})'
            )
        except TransformException as e:
            self.get_logger().warn(f'Could not transform: {e}')

def main(args=None):
    rclpy.init(args=args)
    node = TFListener()
    try:
        rclpy.spin(node)
    except KeyboardInterrupt:
        pass
    node.destroy_node()
    rclpy.shutdown()

if __name__ == '__main__':
    main()
```

### 13.3.5 配置并运行

在 `setup.py` 中添加入口点：

```python
    entry_points={
        'console_scripts': [
            'static_tf_broadcaster = my_tf_pkg.static_tf_broadcaster:main',
            'dynamic_tf_broadcaster = my_tf_pkg.dynamic_tf_broadcaster:main',
            'tf_listener = my_tf_pkg.tf_listener:main',
        ],
    },
```

编译功能包：

```bash
cd ~/ros2_ws
colcon build --packages-select my_tf_pkg
source install/setup.bash
```

然后分别启动三个节点。为了便于管理，我们可以创建一个launch文件，一次性启动所有节点。

## 13.4 使用 Launch 文件启动 TF 节点


修改`setup.py`文件：


## 13.6 TF2 的高级用法

### 13.6.1 查询历史变换

`lookup_transform` 可以指定时间点，例如查询过去某一时刻的变换：

```python
time_point = rclpy.time.Time(seconds=time.time() - 1.0)  # 1秒前
trans = self.tf_buffer.lookup_transform('world', 'laser_frame', time_point)
```

也可以查询两个时间点之间的变换（如果缓存中有）：

```python
trans = self.tf_buffer.lookup_transform_full(
    target_frame='world',
    target_time=rclpy.time.Time(),
    source_frame='laser_frame',
    source_time=rclpy.time.Time(seconds=time.time() - 1.0),
    fixed_frame='world'
)
```

### 13.6.2 变换坐标点

`tf2_geometry_msgs` 提供了将点、向量、姿态等从源坐标系变换到目标坐标系的函数。例如：

```python
from tf2_geometry_msgs import do_transform_point
from geometry_msgs.msg import PointStamped

# 创建一个在 laser_frame 中的点
point_laser = PointStamped()
point_laser.header.frame_id = 'laser_frame'
point_laser.header.stamp = self.get_clock().now().to_msg()
point_laser.point.x = 0.1
point_laser.point.y = 0.0
point_laser.point.z = 0.0

# 查询 world 到 laser_frame 的变换
trans = self.tf_buffer.lookup_transform('world', 'laser_frame', rclpy.time.Time())

# 变换点
point_world = do_transform_point(point_laser, trans)
```

### 13.6.3 使用 TF 静态变换命令行工具

对于简单的静态变换，可以使用 `tf2_ros` 提供的 `static_transform_publisher` 节点，无需编写代码。例如：

```bash
ros2 run tf2_ros static_transform_publisher 0.2 0 0 0 0 0.1745 1 base_link laser_frame
```

其中 `0.2 0 0` 是平移（x, y, z），`0 0 0.1745 1` 是四元数（x, y, z, w），这里绕 Z 轴 10 度。该节点会持续发布静态变换。
